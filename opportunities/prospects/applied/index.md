# Application Sent

Application sent to the prospective company.
The next stage is one of the following:

* Interviewing
    * The application was successfully received and an interview has been set up.
* Retry
    * The application may have been rejected or ghosted but we'll retry later.
* Ghosted
    * The application hasn't received any feedback within ~6 weeks.
* Rejected
    * The application received rejection correspondence.
